# Prerequisites

1.Docker

2.GPU(minimum 2GB memory)

3.NVIDIA GPU + CUDA CuDNN

# Installing and Running

```
$ git clone https://github.com/wb-08/SubVision
$ cd SubVision/build_data
$ docker-compose build
$ docker-compose up
```
